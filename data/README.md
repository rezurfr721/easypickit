# Données POE2

Ce dossier contient les données statiques pour les différentes catégories d'items.
Les fichiers JSON sont mis à jour manuellement depuis l'API poe2scout.com.

## Mise à jour des données

Pour mettre à jour les données :
1. Visiter https://poe2scout.com/api/items/[category]?&league=Standard
2. Copier les données
3. Sauvegarder dans le fichier JSON correspondant

## Structure des fichiers
- currency.json : Monnaies
- breachcatalyst.json : Items Breach
- weapon.json : Armes
- armour.json : Armures
- etc...
